## 快速开始

```
#下载模板
npx degit https://github.com/Jazee6/vitepress-theme-blur

#进入项目
cd vitepress-theme-blur

#安装依赖
pnpm i

#启动项目
pnpm run dev

#编写文章 
#./posts目录使用markdown语法编写文章

#修改配置
#./vitepress/config.mts

#打包上传
pnpm run build
```
